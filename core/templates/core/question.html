How do I center a div in CSS?

Asked by John Doe on May 15, 2024

2 Answers

Jane Smith: You can center a div using flexbox by setting the parent container to display: flex and then using justify-content: center and align-items: center.

Alex Johnson: Another method is to use margin: auto; with a set width for the div. This will center the div horizontally.
